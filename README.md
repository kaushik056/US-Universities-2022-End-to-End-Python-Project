# Project Name: US Universities Data Analysis 2022
This repository contains a Python project focused on the analysis of US university data. The project utilizes popular data manipulation and visualization libraries such as Pandas, Seaborn, and Matplotlib to clean, manipulate, analyze, and visualize the data.

# Overview
The project aims to provide insights into various aspects of US universities, including demographics, enrollment statistics, academic performance, and more. By leveraging Python's powerful data processing capabilities and visualization tools, we explore patterns, trends, and correlations within the dataset.

# Features
- Data cleaning: Handle missing values, remove duplicates, and ensure data consistency.
- Data manipulation: Perform transformations, aggregations, and merges to prepare the data for analysis.
- Data analysis: Explore key metrics, distributions, and relationships within the dataset.
- Data visualization: Create informative visualizations using Seaborn and Matplotlib to convey insights effectively.

# Requirements
To run the project, ensure you have the following dependencies installed:
1. Python
2. Pandas
3. Seaborn
4. Matplotlib

# Usage
Clone the repository: (git clone https://github.com/kaushik056/us-university-data-analysis.git)
Navigate to the project directory
Run the main Python script
Follow the prompts or explore the Jupyter notebooks to interact with the data analysis and visualizations.
